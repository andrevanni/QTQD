Visão geral

O que é o QTQD

O QTQD é um sistema de avaliação financeira semanal. Você lança os valores de QT (Quanto Tenho) — saldo bancário, contas a receber, estoque etc. — e QD (Quanto Devo) — contas a pagar, dívidas, financiamentos — e o sistema calcula automaticamente indicadores como Saldo, Índice QT/QD, PME, Ciclo de Financiamento e Margem Bruta. A análise é semanal e acumula um histórico que alimenta gráficos de tendência e o Inspetor IA.

⚡ Inspetor IA

Análise automática da semana

É a tela de abertura do portal. Mostra um diagnóstico completo da última semana:

  • 4 KPIs de topo: Saldo, Índice, QT Total, QD Total.
  • Semáforo Executivo: 6 indicadores classificados como verde/amarelo/vermelho.
  • Composição QT e QD: barras proporcionais com o peso de cada componente.
  • Gráfico de evolução: Índice + QT + QD + Saldo das últimas 52 semanas.
  • Diagnóstico IA: texto analítico com tendências (regressão linear), projeção 4 semanas e comparação com médias históricas.
  • Riscos e Plano de ação: alertas automáticos e recomendações priorizadas.

Botão "Gerar PDF" imprime a análise completa via janela de impressão do navegador.

📊 Painel de Avaliação

Tabela comparativa semana a semana

Mostra todos os indicadores em colunas (uma por semana, mais recente à esquerda). A primeira coluna (nomes dos indicadores) fica fixa enquanto você rola horizontalmente. Use o filtro "Todos os anos" para limitar a um ano específico, ou o botão "Maximizar" para abrir em tela cheia.

📈 Gráficos

Crie visualizações personalizadas

Use o botão "+ Criar novo gráfico":

  • Dê um nome ao gráfico (ex: "Evolução do Saldo — 12 semanas").
  • Clique nos campos que devem aparecer (na ordem que quiser).
  • Escolha período (semanas/meses/anos), quantidade, tipo (linha/barra) e escala (valor real ou % de variação).
  • Clique em "Salvar Gráfico". Ele aparece na lista e pode ser editado ou reordenado.

Marque "Incluir no relatório PDF" para que o gráfico apareça no e-mail semanal. Os gráficos são compartilhados entre todos os usuários do mesmo cliente.

🕓 Histórico

Todas as semanas registradas

Lista cronológica (mais recente primeiro). Cada linha mostra status, QT, QD, Saldo e Índice. Ações disponíveis:

  • Editar: abre o formulário com os dados preenchidos.
  • Excluir: remove o lançamento (cuidado, ação irreversível).

📝 Lançamentos

Entrada de dados semanal

Formulário principal para registrar uma nova semana ou editar uma existente.

  • Comece pela Data da semana (formato dd/mm/aaaa).
  • Preencha QT, QD e Informações Complementares — campos detalhados (cartões, convênios etc.) somam automaticamente quando preenchidos.
  • Use o botão "⬇ Excel" para baixar um template pré-preenchido com os dados existentes (útil para enviar a alguém preencher offline).
  • Use "⬆ Importar" para subir um Excel preenchido — ele cria/atualiza automaticamente o lançamento como rascunho.
  • Status Rascunho = em edição (não entra em gráficos/inspetor). Status Fechada = consolidada (entra em todas as análises e dispara e-mail).
  • Índice de Faltas: digite o percentual normalmente (6,82 = 6,82%). O sistema converte automaticamente.

📦 Excesso Crítico

Análise de estoque parado por curva

Calcula automaticamente o valor de excesso de estoque por curva (A/B/C/D) a partir do Excel de fabricantes que sai do seu ERP. O resultado pode ser aplicado direto em qualquer semana já cadastrada — substituindo os campos manuais de Excesso Curva A/B/C/D.

Como usar:

  1. Ajuste os limites por curva (em dias de cobertura). Padrão: A=120, B=150, C=150, D=180. Clique "Salvar limites" para persistir a configuração.
  2. Clique "⬆ Importar Excel" e selecione o arquivo do ERP. Formato esperado: colunas Nome Completo, Linha, Curva, Filial, MediaF Un, Qtd Estoque, Estoque Valor.
  3. O sistema mostra 4 cards (A/B/C/D + Total) com o valor calculado e a lista dos top 100 produtos críticos. Use os filtros de curva e busca para investigar.
  4. Em "Aplicar ao Lançamento", escolha a semana e clique no botão. Os 4 valores são gravados no lançamento — visíveis em Painel, Inspetor IA e e-mail.

Regra de cálculo:

  • Cobertura em dias = (Qtd Estoque / MediaF Un) × 30
  • Se cobertura > limite da curva → excesso em unidades = Qtd Estoque − (MediaF Un × limite / 30)
  • Custo unitário = Estoque Valor / Qtd Estoque
  • Excesso em R$ = excesso_un × custo_un
  • Produtos sem venda (MediaF Un = 0 e Qtd Estoque > 1): todo o estoque é considerado excesso.

💡 Boas práticas

Para tirar o máximo do QTQD

  • Frequência semanal: faça o lançamento sempre no mesmo dia da semana — segunda-feira é o mais comum. Consistência é o que permite as análises de tendência.
  • Detalhe sub-itens: em vez de só preencher "Contas a receber" total, detalhe Cartões / Convênios / Cheques. Isso enriquece a composição mostrada no Inspetor IA.
  • Mantenha PMP, PMV e PME atualizados: são a base para o Ciclo de Financiamento. Quando faltam, o ciclo aparece vazio.
  • Use o Excesso Crítico mensalmente: exporte o Excel do ERP no fim do mês e aplique na semana correspondente. Acompanhe a evolução nos gráficos.
  • Feche os rascunhos: só semanas "fechadas" alimentam os gráficos e o Inspetor IA. Rascunhos ficam invisíveis para as análises.
  • Tenha pelo menos 8 semanas: o Inspetor IA precisa de histórico para identificar tendências e fazer projeções confiáveis.
